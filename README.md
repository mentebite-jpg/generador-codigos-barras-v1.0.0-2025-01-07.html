# Generador de Códigos de Barras Profesional

Generador web profesional de códigos de barras con cálculo automático de dígito validador.

## Características
- ✅ 4 formatos: EAN-14, EAN-13, UPC-12, CODE-128
- ✅ Cálculo automático de dígito validador
- ✅ Exportación a SVG, PNG, JPG
- ✅ Rotación y colores personalizables
- ✅ DPI configurable (150, 300, 600)
- ✅ Compatible con Google Sites

## Uso
1. Selecciona el formato deseado
2. Ingresa los datos
3. Configura colores, rotación y escala
4. Descarga en el formato preferido

## Tecnologías
- HTML5, CSS3, JavaScript
- JsBarcode library
- Compatible con todos los navegadores modernos
